Description:
Responsive front-end application with api connection to 'Adzuna' with simple job searching engine.

Use: 
1. Clone project "git clone git@github.com:Maximilianix/ZipTest.git"
2. Open terminal in api folder and run "npm install"
3. From api folder run "node index.js" to start node server
4. Open terminal in app folder and run "npm install"
5. From app folder run "npm run start" to start local server
6. In browser go to "http://localhost:1234/"


Functionality:
1. Fully responsive
2. Search by Title and Location (php & london)
3. Add to favorite option on "star" click (save in local storage)
4. Show modal with a few details on job Title or image click
